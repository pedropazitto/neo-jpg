# Neo.jpg

Resize & compress images instantly. A lightweight PWA.

## Features
- Resize images to configurable max dimension (default: 600px)
- Compress to target file size (default: 80KB)
- Output as JPG, WebP, or PNG
- Native share on mobile
- Works offline
- Batch processing with Download All

## Setup
1. Host these files on any static server (GitHub Pages, Netlify, Vercel, etc.)
2. Visit the URL on your phone
3. Add to Home Screen for app-like experience

## Convert to APK
1. Host the PWA
2. Go to https://pwabuilder.com
3. Enter your hosted URL
4. Download the APK
